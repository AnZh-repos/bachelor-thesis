# Bachelor's Thesis

**In-Depth Research on User Interface and User Experience: What Makes the Design Perfect**

Andrei Zhaliaznou — Vistula University, Warsaw, 2023  
Supervisor: Dr inż. Marcin Wawryszczuk  
Student no. 52202

---

The thesis covers UI/UX theory and practice across five chapters: problem framing,
literature review (Nielsen's heuristics, Fitts' Law, Stanford credibility guidelines),
methodology, user-centred design principles with case studies (healthcare.gov, Myspace,
AOL), and results from original research on what drives user satisfaction in web design.
